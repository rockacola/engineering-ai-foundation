# Known Issues

## Summary

Active known issues and limitations in the Engineering AI Foundation.

## Key Facts

- `templates/` does not include pre-populated `.ai/context/` or `.ai/agents/` files — those are project-specific and must be generated by the operator after bootstrap. The bootstrap workflow handles directory creation from `spec/repo-structure.yaml`.
- `spec/agent-registry.yaml` does not distinguish which agents are optional vs. required for different repo types, leaving that judgement to the operator.
